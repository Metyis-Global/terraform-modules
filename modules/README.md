# Introduction 
This is set of Terraform modules that describes how infrastructure must be created according to needs of Eruvaka and Azure best practices
These modules are support terraform ~> 3.17.0

It includes:
- Private endpoint
- Private DNS zone
- Network security group
- Initial infra as VNET + resource group
- Subnet
- Key vault
- Storage account
- Log analytics workspace
- WAN, Hub, PTS VPN gateway
- Event Hub namespaces with event hubs (topics) with and without captures
- IoT hub
- DPS IoT hub
- Secrets
